#Lecture 3_0
■ Material 생성
Material (Inputs)
- Material 입력은 Material의 특성에 따라 변함
- 회색으로 표시된 입력은 현재 특성이라면 사용이 불가능 함을 의미함
- 자동으로 특성에 따라 처리됨
=> 객체의 표면이나 모양을 표현하기 위한 데이터

베이스 컬러
- 색에 관련된 물리적인 특성을 가짐
- R, G, B 값을 입력으로 함
- 각 채널 별로 0.0~1.0 사이의 값
- 더 크거나 작으면 clamp 됨
=> 월드에 세팅된 조명, 조명 볼륨, 환경맵 등의 영향을 받는다.
=> Material Domain이 Surface이고, Shading Model이 Default Lit인 경우 위처럼 영향을 받음 

Metallic
- 얼마나 금속성이 있는가를 설정
- 0.0~1.0 사이의 값
- 광선의 반사율 및 정확도를 값 하나로 제어
=> 어두운 부분과 밝은 부분이 두드러짐

스페큘러
- 메탈릭 값이 1.0보다 작을 경우에만 효과가 있음
- 0.0~1.0 사이의 값
- 물체의 반사성 양을 조절하는 데 쓰임
- 보통은 메탈릭 값만 사용함

러프니스
- 표면의 거칠기를 제어
- 0.0~1.0 사이의 값
- 반사되는 광선을 얼마나 희미하게 할 것인지를 결정함
- 거울일 경우 0, 나무일 경우 1에 가깝게
- 러프니스를 설정하게 되면 주변 환경이 반사되는 효과가 나타남!

이미시브 컬러
- 스스로 빛을 내뿜으며, 외부 조명의 영향을 받지 않음
- 발광색을 설정할 수 있음
- HDR(High Dynamic Range) 기반
- 1.0보다 큰 값이 허용됨
- 주로 파티클이나 네온사인과 같이 발광하는 효과에 사용
- Bloom(빛 퍼짐 현상)에 영향을 주기 때문에 빛이 퍼져 나가는 렌더링 결과를 줄 수 있음
=> TV나 동영상과 같은 것을 표현할 때 이용

오파시티
- 얼마나 투명한지를 결정
- 0.0~1.0 사이
- 0.0일 경우 완전 투명, 1.0일 경우 완전 불투명
=> 오파시티를 적용하려면 Blend Mode가 Translucent로 설정되어야 함
=> Metaillic, 스페큘러, 러프니스 등을 같이 사용하려면 Translucency 탭에서 라이팅 모드를 Surface가 들어간 걸로 설정해주어야 함
=> Alpha Blend : 색을 섞어서 뒤의 물체가 보임
=> Alpha가 존재하면 Depth Buffer Culling을 방해하여 성능이 저하됨
