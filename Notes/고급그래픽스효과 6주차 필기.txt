■ Material (Inputs 3)
서브서피스(Sub Surface Scattering) 컬러
- 재질 표면 아래의 색을 표면
- 피부의 붉은 빛깔
- 얼음의 푸르스름한 빛깔
- 도달한 광선의 색이 변함(내부 굴절 후 기본 광선과 혼합)

실습
- 셰이딩 모델 프로퍼티 → Subsurface
- 서브서피스 컬러 부여

Scattering (빛의 산란)
- Ray Tracing

Opacity : (셰이딩 모델이 Subsurface일 때인 것 같음) 빛이 얼마나 잘 퍼지느냐
- 0일수록 잘 퍼짐, 1일수록 안 퍼짐

엠비언트 오클루전
- 셀프 쉐도우 표현, 미세한 광량 차이, 정밀한 환경망 표현
- 표면의 작은 틈에 생기는 그림자
- 오브젝트 단위/씬 단위로 적용하는 두 가지 방법이 존재함

실습
- 앰비언트 오클루전 적용

러프니스
- 거칠기를 표현
- 실습에서는 텍스처의 R채널을 활용



#Lecture 5_1
굴절
- 밀도 차에 의한 굴절 현상 표현
- 투명한 물체에 적용
- 투명한 물체에서 빛의 굴절률에 따른 왜곡 현상을 나타내기 위해 사용

실습
- 굴절 적용
- Fresnel 노드 적용
=> 베이스 컬러가 반드시 적용되어야 함(광원 연산이 적용되야 하기 때문)

Surface Translucency - 굴절을 표현할 때 사용함

꿀렁거리게 하는 법?
1. Normal Map : 실제 버텍스의 좌표는 바뀌지 않지만 법선벡터를 좌우로 변형하며 튀어나온듯한 느낌을 줌 (time 노드를 통해 uv를 이동시킴)
2. 월드 포지션 오프셋, 테셀레이션 멀티플라이어 등을 이용한 Displace Mapping
=> 출렁거리는 물을 표현하기 위해서는 위의 두 가지를 모두 사용하는 것이 좋다.

Panner 노드 : uv 좌표를 생성하고, 축별 속도를 주어 이동이 가능하게 한다.

찰랑거리는 물 표현을 위한 머터리얼 생성
- Texture Coordinate Animation : Panner Node
- Fresnel Node
- 테셀레이션 멀티플라이어
