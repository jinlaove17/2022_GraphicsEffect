#Lecture 1_1
그래픽스 : 빛과 오브젝트 간 상호작용

■ 라이트 기본
가장 많이 쓰이는 라이트 종류 세가지
- Directional Light
- Point Light
- Spot Light
- Rect Light : LED Light를 구현하기 위해 생겨남

방향성 조명(Directional Light)
- 무한히 먼 거리에 있는 광원(위치)에서 오는 빛을 흉내 내는 것
- 모든 광선이 평행함
- 보통 태양광을 나타낼 때 쓰임
- (그림 참고)
- 태양이 실제로 엄청나게 크기 때문에 태양광선은 거의 평행하게 지구에 도달함
=> 모든 월드에서 1개만 사용할 것
=> 회전의 영향만 받는다.

점 조명(Point Light)
- 하나의 점으로 표현되는 광원에서 나오는 빛을 흉내 내는 것
- 포인트 라이트의 경우 구체가 UI적인 표현으로 사용됨
=> 영역이 겹치도록 사용하지 말 것
=> 위치의 영향만 받는다.

스팟 조명(Spot Light)
- 하나의 점으로 표현되는 광원에서 원뿔 모양으로 나오는 빛을 흉내내는 것
- 보통 손전등, 책상등, 전조등 등을 표현할 때 쓰임
=> 회전과 위치 모두 영향을 받는다.

■ 실습 : 언리얼에서 라이팅 적용
- 액터(Actor) : 씬에 배치될 수 있는 모든 것



#Lecture 2_0
- Graphics : 빛과 객체 간의 상호작용을 만드는 것

■ illumination Model
- 빛과 물체와 눈의 상호작용을 표현하는 모델
- 3가지 필요 요소 : Light Source, Object, Camera
- Ambient(환경광), Specular(정반사), Diffuse(난반사)

Ambient(환경광)
- 씬 전체에 같은 값으로 표현되는 광원 효과
- 빛의 방향과는 관계 없이 해당 물체에 미치는 주변광의 합으로 나타냄

Diffuse(난반사)
- 광선이 물체 표면에서 완전한 난반사를 함
- 표면에 떨어지는 광선의 양에 따라 강도가 정해짐
=> 카메라의 위치가 다르더라도 세기는 동일

Specular(정반사)
- 광선이 물체 표면에서 정확한 반사를 함
=> 카메라의 위치에 따라 세기가 달라짐

Amibent, Diffuse, Specular는 아래 식과 같이 값을 기반으로 사용됨
- (그림 참고)

UE4는 물리기반 셰이딩 형태를 따름
- 바탕색, 거칠기, 금속성, 반사성

■ 머터리얼(Material)
오브젝트에 특성을 부여하는 것
- 표면 질감, 색, 텍스쳐, ...
