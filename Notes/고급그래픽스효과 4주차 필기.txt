#Lecture 3_1
■ Material(Inputs)
오파시티
- 얼마나 투명한지를 결정
- 0.0~1.0 사이
- 0.0일 경우 완전 투명, 1.0일 경우 완전 불투명
=> 오파시티를 적용하려면 Blend Mode가 Translucent로 설정되어야 함
=> Metaillic, 스페큘러, 러프니스 등을 같이 사용하려면 Translucency 탭에서 라이팅 모드를 Surface가 들어간 걸로 설정해주어야 함
=> Alpha Blend : 색을 섞어서 뒤의 물체가 보임
=> Alpha가 존재하면 Depth Buffer Culling을 방해하여 성능이 저하됨
=> 블렌더 모드가 켜지면은 이미시브 컬러가 베이스 컬러로 작용한다.(반투명한 물체들은 빛을 투과하기 때문)

오파시티 마스크
- 오파시티 마스크는 0(완전 불투명)아니면 1(완전 투명)로 나뉨
- Depth Culling에 지장을 주지 않아 성능저하가 발생하지 않음
=> 오파시티 마스크를 사용하려면 Blend Mode를 Masked로 설정해야 한다.

노멀
- 표면에 방향성을 부여

- R, G, B 값을 입력으로 함
- 3차원 Vector(표면의 방향을 표현하기 위함)

Texture 좌표
- 사각형의 Texture를 기본으로 함
- Texture 좌표는 0.0~1.0 사이로 정규화 되어 있음
- 좌표는 컬러 형식으로 확인 가능
- 우하향 좌표계((0, 0)이 좌측 상단, (1, 1)이 우측 하단)
- uv 좌표계(u는 r채널(x축), v는 g채널(y축))
- 매우 많은 용도로 사용될 수 있음(ex : 방향성을 가지는 텍스처 생성, 텍스처를 입힐 때 해상도 조절, 텍스처 반전을 위한 좌표 생성, ...)

■ Linear Interpolation 노드
- 보간(Interpolation)은 머터리얼 제작에서 중요한 기능 중 하나
- A 값과 B 값이 주어졌을 때, 두 값 사이에 존재하는 보간 값을 얻어 옴

여러가지 보간 방법이 존재함
- Nearest Neighbor
- Linear

선형 보간(Linear Interpolation)
- A, B, Value 세 값을 입력으로 받음
- Value는 0.0~1.0 사이의 값
- Result = A * (1.0 - Value) + B * Value
- 즉, A와 B 사이에서 Value 만큼의 가중치를 주어 값을 계산하는 형식
